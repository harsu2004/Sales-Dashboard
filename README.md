1.  Project Title
Sales Dashboard Analysis

2. What the Dashboard Does & Why It Exists
  - This project presents an interactive Sales Dashboard designed to analyze and visualize key business metrics such as total sales, profit,     units sold, and cost of goods sold (COGS).

The purpose of this dashboard is to help businesses and stakeholders:
  - Understand sales performance over time
  - Identify high-performing products
  - Analyze profit distribution across segments
  - Make data-driven decisions

3. Key Technologies Used
    - Power BI – for dashboard creation
    - Microsoft Excel – for data preprocessing
    - Data Visualization Techniques – charts, graphs, KPIs
    - Basic Data Analysis Concepts – aggregation, filtering, segmentation

4. Key Highlights of the Dashboard:

    - Displays Total Sales, Profit, Units Sold, and COGS in KPI cards
    - Monthly Sales Trend Analysis to track performance over time
    - Sales by Segment (Government, Enterprise, etc.) using a donut chart
    - Top Products by Profit to identify best-performing items
    - Profit by Discount Band to understand impact of discounts
    - Interactive Filters (Year-wise) for dynamic data exploration
